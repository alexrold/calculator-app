# Calculadora en React Native

Una aplicación de calculadora simple construida con React Native y Expo.

## Características

- **Operaciones Aritméticas:** Suma, resta, multiplicación y división.
- **Cálculo en Cadena:** Realiza operaciones secuenciales (ej. `10 + 5 - 3 = 12`).
- **Manejo de Errores:** Muestra un mensaje de error claro en la división por cero.
- **Funciones Básicas:** Incluye cambio de signo (`+/-`), borrado de último dígito (`del`) y limpieza total (`C`).
- **Pruebas Unitarias:** Lógica de negocio cubierta por un conjunto de pruebas con Jest.

## Cómo Empezar

### Prerrequisitos

- Node.js
- pnpm (o el gestor de paquetes de tu elección)
- Expo Go (en tu dispositivo móvil) o un emulador de Android/iOS.

### Instalación y Ejecución

1.  **Instalar dependencias:**

    ```bash
    pnpm install
    ```

2.  **Iniciar el servidor de desarrollo:**

    ```bash
    pnpm start
    ```

    o

    ```bash
    npx expo start
    ```

3.  **Ejecutar las pruebas:**
    ```bash
    pnpm test
    ```

Una vez iniciado el servidor, escanea el código QR con la aplicación Expo Go en tu teléfono o selecciona una de las opciones para abrirlo en un emulador.
